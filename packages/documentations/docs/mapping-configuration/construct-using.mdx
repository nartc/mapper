Call `constructUsing()` and pass in a `DestinationConstructor` to customize how AutoMapper should construct the `Destination` before every map operation against that `Destination`.

```ts
createMap(
    mapper,
    User,
    UserDto, // ðŸ‘ˆ --the destination modifier--ðŸ‘‡
    constructUsing((sourceObject, destinationIdentifier) => {
        // sourceObject is the data when run: mapper.map(sourceObject...);
    })
);
```

:::info

All built-in `MappingStrategy` should have a default `destinationConstructor` (e.g: `@automapper/classes` default is `new Destination()`).

:::
